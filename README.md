# Crashcourse Machine Learning: From linear regression to implementing neural networks in Pytorch

## Description

In this session, participants build a foundation in data-driven modeling, from linear regression to implementing neural networks in PyTorch. Using the relationship between satellite-based LAI and ERA5 meteorology as an example, we explore whether this relationship can be understood and modeled with data-driven approaches.

The session introduces key concepts such as overfitting, underfitting, and validation on a test set, focusing on what to consider when developing a data-driven modeling pipeline. Through hands-on experimentation, participants test different modeling choices and build a regressor from meteorology to LAI, aiming to achieve the best possible predictive performance into the future. By the end, participants will understand core modeling pitfalls and be able to implement and train a neural network in PyTorch.
